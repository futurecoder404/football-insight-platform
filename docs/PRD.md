# Football Insight Platform

## Vision

A real-time football analytics platform that helps fans understand not only what is happening in a match but why it is happening.

## Core Features

### Teams
- Team details
- Manager
- FIFA ranking
- Squad

### Players
- Player profile
- Position
- Club
- Statistics

### Matches
- Fixtures
- Results
- Live scores

### Match Events
- Goals
- Yellow cards
- Red cards
- Substitutions

### Predictions
- Win probability
- Draw probability
- Loss probability
- Explanation of prediction

### Future Features
- Match momentum
- AI match insights
- Real-time pitch visualization
